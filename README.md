# 📑 App Data Manager

A simple browser-based application for managing and storing app credentials locally using **HTML, CSS, and JavaScript**.  
This tool allows users to save, view, download, and clear their app login details directly in the browser with **LocalStorage**.

---

## 🚀 Features
- **Add Credentials**: Save app name, username, and password.
- **Persistent Storage**: Data is stored in the browser’s LocalStorage.
- **Credential List**: View all saved credentials in a clean card layout.
- **Download as Text File**: Export all saved credentials into a `.txt` file.
- **Clear Storage**: Delete all saved credentials from LocalStorage with one click.
- **Responsive UI**: Modern, minimal design with responsive layout.

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the app
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic for data handling, storage, and file download
- **LocalStorage API** – Persistent storage in the browser

## ⚙️ How It Works
1. Enter **App Name**, **Username**, and **Password** in the input fields.
2. Click **Add to List** to save the entry.
3. Credentials are stored in **LocalStorage** and displayed in the list.
4. Use **Download All as .txt** to export credentials.
5. Use **Clear Local Storage** to remove all saved data.
